==============================================
 BOT FISCAL - INÍCIO RÁPIDO
==============================================

1. Instale as dependências
   cd backend
   npm install

2. Copie e ajuste as variáveis de ambiente
   cp .env.example .env
   # Preencha WHATSAPP_TOKEN, PHONE_NUMBER_ID, VERIFY_TOKEN, APP_BASE_URL

3. Aplique as migrações e seeds
   npx prisma migrate dev --name whatsapp-bot
   npx ts-node prisma/seed.ts

4. Configure o webhook no painel da Meta
   Callback URL: http://localhost:3000/webhook
   Verify token: (mesmo valor configurado no .env)

5. Adicione os testers e envie a primeira mensagem pelo número de teste.

6. Inicie o servidor
   npm run dev

7. Acompanhe os testes
   - Dashboard: http://localhost:3000/dashboard
   - Health check: http://localhost:3000/api/health

Comandos úteis no WhatsApp:
  novo | nova empresa | pendencias | resumo | problema | observacao | desabafo | pular | retomar | encerrar

Fluxos disponíveis:
  - Simples Nacional (comércio com movimento)
  - Lucro Presumido (comércio com movimento)

Lembretes automáticos D-7/D-3/D-1 são enviados ao preparador cadastrado.
